## How to Install
- Using `npm` from npm registry:
```
npm i -g lombokts-cli
```

## How to Use
### NESS (NodeJS + Express + Sequelize + Swagger)
**Initialize new NESS project**
```
lts ness init <appName>
```
This will generate a new folder named `<appName>` in your current directory.  Make sure to change `<appName>` to your actual project name and that there is no preexisting folder with the same name as your project name.
